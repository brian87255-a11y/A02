-- Creating your first repo -- \
  -- Setting up account -- \
Step 1: Go to https://github.com/ \
Step 2: Locate the "Sign up for GitHub" button \
Step 3: On the open space fill out the necessary information then click the button \
Step 4: Once on the new page, create a password, a username, and then select your region \
Step 5: After filling out the necessary information click "Create account" \
Step 5: Verify the email \
  -- Creating the repo -- \
Step 1: Once you are logged in, locate the "Create repository" button \
Step 2: After clicking the button, you will give the repo a name \
Step 3: Scrolling down the page, you can give the repo a description, this not necessary but useful recalling about the repo \
Step 4: Under Configuration, you wil need to choose visibility, select public or private to your liking \
Step 5: Finalizing the repo you can add a README file, .gitignore and license. These are extra and just an addition to the repo \
Step 5: Lastly, you will clcik "Create repository" \
** Your Repo is now successful create ** \
